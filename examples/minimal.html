What html2pdf keeps

Screen styles, not print

By default this card keeps its colored background, soft shadow, and rounded corners. The default @media print rules in most browsers would strip them — html2pdf uses emulateMediaType('screen') instead.

emulateMedia: screen

Sticky elements disabled

Sticky nav bars and timeline labels would normally repeat on every page of a long document. html2pdf disables position: sticky before capture, so each element only appears once.

no-repeat sticky
